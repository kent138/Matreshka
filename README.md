# МАТРЁШКА — интернет-магазин тканей и фурнитуры

Современный интернет-магазин тканей, фурнитуры и аксессуаров для рукоделия из города **Чита** с доставкой по всей России.

Фирменный стиль: красный / чёрный / белый, логотип — красный клубок ниток с иглой. Адаптивная вёрстка, PWA, SEO-разметка.

## Запуск

Это статический сайт без сборки. Достаточно отдать папку любым статическим сервером (нужен HTTP, а не `file://`, чтобы работал Service Worker и `BroadcastChannel`):

```bash
# из корня проекта
python3 -m http.server 8080
# затем откройте http://localhost:8080
```

Любой другой статический сервер (`npx serve`, nginx, GitHub Pages и т.д.) тоже подойдёт.

## Технологии

- Чистые **HTML + CSS + JavaScript** (без фреймворков и сборки)
- Хранение данных в `localStorage` (товары, пользователи, корзина, избранное, заказы)
- Реальное время заказов в панели модерации через `BroadcastChannel` (между вкладками одного браузера)
- PWA: `manifest.json` + Service Worker (`sw.js`) — офлайн-режим, установка на устройство

## Структура

```
index.html          — оболочка приложения (шапка, навигация, футер, модалки)
css/styles.css      — дизайн-система и все стили
js/data.js          — каталог-«семена», категории, цвета, статьи блога, пароль админа
js/store.js         — слой данных: localStorage, авторизация, корзина, заказы, события
js/ui.js            — UI-хелперы: деньги, тосты, модалки, расчёт доставки, карточки товаров
js/admin.js         — админ-панель каталога и панель модерации заказов
js/app.js           — роутер и все экраны (главная, каталог, товар, корзина, оформление, кабинет и др.)
assets/logo.svg     — логотип (клубок ниток с иглой)
manifest.json, sw.js— PWA
robots.txt, sitemap.xml — SEO
```

## Возможности

**Покупателям**
- Каталог с фильтрами (категория, цена, цвет, тип ткани), поиском и сортировкой
- Карточки товаров с фото, составом, шириной, плотностью, рекомендациями по уходу
- Отрез ткани нужной длины (от 0,5 м, шаг 0,1 м)
- Корзина, избранное
- Оформление заказа: расчёт доставки по региону (СДЭК, Почта России, самовывоз), бесплатная доставка от 5000 ₽
- Оплата: картой онлайн (МИР), СБП, наличными при получении (без Visa/MasterCard)
- Регистрация/вход, личный кабинет: заказы, отслеживание статуса, адреса, бонусы, настройки
- Чат поддержки, страницы «Доставка», «Возврат», «О нас», «Контакты» с картой, FAQ, блог

**Администрирование** (меню «три точки» в правом верхнем углу, пароль `7316`)
- **Редактировать каталог:** добавление/редактирование/удаление товаров, загрузка фото (drag-and-drop или выбор файла), управление остатками, изменение порядка товаров перетаскиванием
- **Вход в модерацию:** все заказы поступают в реальном времени, изменение статуса (новый → в обработке → отправлен → доставлен / отменён), поиск по номеру/имени/телефону/дате, фильтр по статусу, экспорт в CSV
